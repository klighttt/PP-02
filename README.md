## Описание

Данное программное средство предназначено для автоматизации ключевых процессов работы пункта выдачи заказов (ПВЗ) Ozon. Программа выполняет учет поступающих заказов, их хранение по ячейкам, поиск и выдачу клиентам, а также контроль просроченных заказов и формирование отчетности.

Разработка выполнена в рамках производственной практики ПП.02 по специальности 09.02.07 «Информационные системы и программирование» на базе действующего ПВЗ Ozon.

Система работает локально, все данные хранятся в файлах JSON и CSV, что обеспечивает автономность работы и безопасность информации о клиентах и заказах.

## Возможности программы

1.Учет заказов
-Приемка заказов от курьеров с присвоением номера

-Автоматическое распределение заказов по свободным ячейкам хранения

-Сохранение данных клиента (ФИО, телефон, дата поступления)

2.Поиск и выдача
-Быстрый поиск заказа по номеру или ФИО клиента

-Оформление выдачи с фиксацией даты и времени

-Автоматическое освобождение ячейки хранения

-Имитация отправки SMS-уведомления клиенту

3.Контроль просрочек
-Автоматическая проверка срока хранения заказов (3 дня)

-Выявление просроченных заказов

-Формирование актов возврата товара на склад

4.Отчетность и аналитика
-Журнал приемки заказов (CSV)

-Журнал выдачи заказов (CSV)

-Статистика по заказам за день/неделю

-Экспорт данных в Excel (CSV) и JSON

-Автоматическое создание резервных копий базы

5.Хранение данных
-Локальная база данных на основе JSON

-История операций в CSV-файлах

-Раздельное хранение отчетов по папкам

-Не требует интернета и сервера
